プライバシーポリシー

本アプリ（以下、「本アプリ」）は、ユーザーのプライバシーを尊重し、以下の方針に基づいて個人情報を取り扱います。

1. 収集する情報

本アプリは、ユーザーの個人情報を収集しません。ただし、アプリの動作に必要な情報（例：設定データや利用状況の統計情報など）をデバイス上に保存することがあります。

2. 情報の利用目的

本アプリは、収集した情報を以下の目的で利用する場合があります。

  • アプリの機能提供および改善
  • ユーザーサポートの提供

3. 情報の共有

本アプリは、ユーザーの個人情報を第三者と共有することはありません。

4. 外部サービスの利用

本アプリは、Microsoft Storeやその他の外部サービスを利用する場合があります。それらのサービスのプライバシーポリシーについては、各サービス提供者の規約をご確認ください。

5. 問い合わせ先

プライバシーポリシーに関するご質問がある場合は、以下の連絡先までお問い合わせください。
